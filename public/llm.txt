# Pontim - Open-Source Scrum Poker Platform

## Overview
Pontim is a modern, open-source Scrum Poker platform designed for agile teams to estimate story points collaboratively in real-time. Built with Next.js, React, and PostgreSQL, Pontim offers an engaging and efficient way to conduct Planning Poker sessions using Fibonacci numbers.

## Key Features
- **Real-time Voting**: Live collaborative estimation with instant results
- **Interactive Experience**: Sound notifications to grab teammates' attention
- **Fibonacci Sequence**: Standard agile estimation using Fibonacci numbers
- **Customizable Rooms**: Create private rooms with team-specific settings
- **Multi-plan Support**: Free and premium plans with different room limits
- **Google OAuth**: Seamless authentication with Google accounts
- **Progressive Web App**: Works on desktop and mobile devices
- **Open Source**: MIT licensed with active community development

## Technology Stack
- **Frontend**: Next.js 14, React 18, TypeScript
- **Backend**: Next.js API routes, Prisma ORM
- **Database**: PostgreSQL
- **Authentication**: NextAuth.js with Google OAuth
- **Payments**: Stripe integration
- **Real-time**: Server-Sent Events (SSE)
- **Styling**: Tailwind CSS
- **Deployment**: Docker support
